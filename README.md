Kelebihan : 
- Searching Movie
- Responsive
- Pagination
- terdapat Page Error
- Terdapat Slider

Dikerjakan menggunakan
- HTML
- CSS
- Bootstrap
- JavaScript
- JQuery
- PHP
- Slider menggunakan Owl Carousel
